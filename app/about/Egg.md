### 说了别点了还要点

朋友们好啊！我是开关机工程师、插拔网线门掌门人。

刚才有个朋友问我：“老师发生什么事了？”我说怎么回事，给我发了几张监控截图。我一看！嗷！原来是昨天，有两个小年轻，三十多岁。一个写 Java 的，重构过微服务；一个写前端的，自封全栈。

他们说，唉……马老师，我这业务在容器里跑着，出口路由断断续续的。老师你能不能教教我“网络功法”，教教我怎么绕过策略，解决我的出口阻断 。 我说可以。我说你在应用层练死代码、堆业务逻辑，不好用。他不服气。我说小朋友你两个手来调我一个“防火墙黑名单”，他找不到。他说你这也没用。我说我这个有用，这是“化劲儿”，传统开发是讲化劲儿的，四两拨千金。几百个G的 DDoS，动不动我这一个 Cloudflare 5 秒盾，全部化掉。

啊……哈！他非要和我试试。我说可以。诶……我一说，他啪就站起来了，很快啊！然后上来就是一个 `nmap` 扫端口，一个 `curl` 刷接口，一个 `ssh` 暴力破解。我全部防区去，防出去了啊！

防出去以后，自然是传统运维点到为止。我 [fail2ban](https://www.fail2ban.org) 已经封到他 IP 上了，没点“永久禁封”。我笑一下准备收手，由这时间，按传统运维的点到为止，他已经输了。如果我这一手发力，一个 `iptables` 规则就把他全段 DROP 掉了。我封在 IP 上没有点确认，他也承认，我先拦截了。

他不知道我阻断已经配好了，他承认我先拦截了。啊，我收手的时间不打了，他突然袭击，用一个“带鉴权漏洞的后台白名单”来勾引我。啊，我大意了啊，没有闪。哎……他的未授权请求给我的库，啊，数据库，黑了一下。

但没关系啊！他也说，啊他日志也说了，两分多钟以后，当时我冷汗就流下来了。连上控制台，我说停停！然后两分钟以后，诶，我回滚了配置，服务器就好了。

我说小伙子你不讲武德，你不懂。他说：“老师对不起，我不懂规矩。”啊，我是……他说他是乱写的。他可不是乱写的啊！假装不懂、骗改配置、白名单偷袭，训练有素！后来他说他练过三四年社工和渗透，啊，看来是有备而来！

这两个年轻人不讲武德，来骗，来偷袭，我一个只会开关机、插拔网线的老同志。这好吗？这不好。我劝这位年轻人耗子尾汁，好好反思，以后不要再犯这样的聪明，小聪明。

啊，呃…… 互联网要以和为贵，要讲武德，不要搞这种窝里斗。谢谢朋友们！
